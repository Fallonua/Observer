Музыкальный клип (Компонент 2)
================================
Для воспроизведения звука поместите файл vivo.mp3 в одну из папок:
- src/main/resources/vivo.mp3
- корень проекта (kurs_writers/vivo.mp3)

Либо укажите полный путь к MP3-файлу в коде ObserverApp (переменная mediaPath).
Без файла приложение запускается; кнопки воспроизведения не дадут звука.
